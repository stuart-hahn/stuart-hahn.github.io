How to Reverse a String in Javascript

Note: This problem and other common coding interview questions (along with their solutions) can be found in this Github Repo

Instructions: Take a string and return a new string with a reversed order of characters. For example:

reverseString('hello') //=> 'olleh'

Ways to Solve This

This is a simple problem that can be solved using a variety of approaches.

The simplest solution would be to use built-in JavaScript methods, but it’s also simple enough to solve this even if the interviewer wants you to avoid built-in methods.

These are a few of the strategies you could use:

  1. Use built-in JavaScript methods
  2. Use a for loop
  3. Use recursion

Built-in JavaScript Methods

JavaScript has built-in methods that can handle things like turning a string into an array, reversing an an array, and turning an array into a string.

Unfortunately, the reverse method doesn’t work directly on strings, so it is necessary to first split the string into an array in order to reverse it with the reverse method.

The good news is that methods can be chained together, so the full solution using JavaScript’s built-in methods is still a simple one-liner.

function reverseString (str) {
  return str
    .split('')
    .reverse()
    .join('')
}

Here’s exactly what’s happening:

  1. str.split("") is taking the str string and splitting it into an array. The argument passed represents what character to split the string on, and in this case I’m passing in an empty string so that it splits on every character.

  2. .reverse() gets chained on and reverses the array that was made by the .split("").

  3. .join("") takes that newly reversed array and joins it back into a string. The argument of an empty string is once again passed, letting the method know to join the letters together without anything between them.

Using a Loop

The built-in methods make up the simplest solution, but sometimes the interviewer might want you to tackle problems in a different way. For example, they might specifically request that you solve this problem without using reverse().

One way to do this is with a for loop.

// 1. create a variable to store the reversed word
let reversed = ''

// create a for loop
// 2. initialize with a value of the length of the string minus 1
// 3. run while i is greater than or equal to zero
// 4. decrement by one on every loop
for (let i = str.length - 1; i >= 0; i--) {
  // 5. add each letter (in descending order) to the 'reversed' variable
  reversed += str[i]
}

// 6. return the 'reversed' variable
return reversed

Here’s that same code without the comments (for easier reading):

function reverseString (str) {
  let reversed = ''

  for (let i = str.length - 1; i >= 0; i--) {
    reversed += str[i]
  }

  return reversed
}

This method really only requires you to know how to create a for loop. You could write it as a decrementing for loop like I’ve done above, but you could also write it as an incrementing for loop with just a couple changes:

function reverseString (str) {
  let reversed = ''

  for (let i = 0; i < str.length; i++) {
    reversed = str[i] + reversed
  }

  return reversed
}
